# **App Name**: AI-Powered Biomedical Portfolio

## Core Features:

- Interactive CV Display: Showcase your CV details in a dynamic, engaging format, with expandable sections for each experience.
- Portfolio Showcase: Display portfolio projects with high-quality images, descriptions, and links to live demos or repositories. Sort and filter by project type and technologies used.
- Personalized Recommendation Tool: A generative AI-powered tool that suggests projects from your portfolio based on a visitor's indicated interests. The tool uses reasoning to highlight relevant projects. Requires the visitor to answer a small questionnaire about his/her interests.
- Contact Form with AI Assistant: Implement a contact form that uses an AI tool to parse and categorize incoming messages (e.g., job offer, collaboration, question) before you review them. This saves time.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and intelligence.
- Background color: Very light grey (#F0F2F5), almost white, for a clean, modern look.
- Accent color: Teal (#009688) to highlight interactive elements and call-to-actions, adding a tech-savvy feel.
- Headline font: 'Space Grotesk' sans-serif for a modern, techy feel, and 'Inter' sans-serif for body text to ensure readability.
- Use a set of minimalist line icons to represent skills, technologies, and portfolio categories.
- Design a clean, grid-based layout with ample whitespace to ensure readability and visual appeal on all devices. Implement a sticky navigation bar for easy access to key sections.
- Use subtle, smooth transitions and animations for page navigation and element interactions to enhance user experience. Employ scroll-triggered animations to reveal content as users scroll down the page.
